# Radial Menu UI

A radial (circular) navigation menu built using only HTML and CSS.  
This project demonstrates how modern interactive UI components can be created without JavaScript using CSS positioning and transitions.

---

## Live Demo

https://sindhusrirani.github.io/menu-UI/

---

## Overview

This project implements a radial menu that expands from a central toggle button.  
Each menu item is positioned in a circular layout and becomes visible when the center button is activated.

The design focuses on simplicity, responsiveness, and smooth UI interaction using only CSS.

---

## Features

- Radial circular navigation layout
- Toggle-based expand and collapse functionality
- Pure HTML and CSS implementation (no JavaScript)
- Hover tooltips for menu items
- Smooth transitions and scaling effects
- Lightweight and fast-loading UI

## How It Works

- A hidden checkbox input controls the menu state
- When the checkbox is checked, CSS transforms position each menu item around a central point
- Each item uses translate transformations to form a circular layout
- Hover states are used to display tooltips
