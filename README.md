#  Radial Menu UI

A creative and interactive **radial menu UI** built using only **HTML and CSS**.  
This project demonstrates modern UI design using circular layouts, hover tooltips, and smooth transitions — without any JavaScript.

---

##  Features

-  Fully functional radial (circular) menu
-  Built with only HTML + CSS (no JavaScript)
-  Click-to-expand center button
-  Hover tooltips for each menu item
-  Clean dark UI design
- Lightweight and responsive-friendly structure

---

##  Tech Stack

- HTML5
- CSS3
